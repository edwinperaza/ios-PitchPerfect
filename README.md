# ios-PitchPerfect

> What's Pitch Perfect?

This is an iOS app that you can record audio and play it in several ways.

> What are the additional features?

* User can save audio
* User can play audio:
  - Slow
  - Fast
  - High Pitch
  - Low Pitch
  - Echo
  - Reverb
